## Section 1: Logic and Basic Propositions

1. Logic and Basic Propositions (Easy)
1. What is logic primarily concerned with?
A) Emotions
B) Valid reasoning
C) Art
D) Politics
Answer: B

2. A proposition is a statement that is:
A) Always true
B) Always false
C) Either true or false
D) Neither true nor false
Answer: C

3. Which symbol represents conjunction in symbolic logic?
A) ∨
B) ∧
C) ¬
D) →
Answer: B

4. The symbol '∨' represents:
A) Negation
B) Conjunction
C) Disjunction
D) Implication
Answer: C

5. A simple proposition contains:
A) Multiple subjects
B) One subject and one predicate
C) No predicate
D) Multiple predicates only
Answer: B

6. Which of the following is NOT a proposition?
A) The sky is blue
B) 2 + 2 = 4
C) Please close the door
D) Paris is in France
Answer: C

7. The negation symbol in logic is:
A) ∧
B) ∨
C) ¬
D) →
Answer: C

8. "If P then Q" is an example of:
A) Conjunction
B) Disjunction
C) Negation
D) Implication
Answer: D

9. A compound proposition contains:
A) Only one simple proposition
B) Two or more simple propositions
C) No propositions
D) Only predicates
Answer: B

10. Logic helps us distinguish between:
A) Good and bad arguments
B) Beautiful and ugly things
C) Rich and poor people
D) Young and old ideas
Answer: A

## Section 3: Basic Fallacies

3. Basic Fallacies (Easy)
21. A fallacy is:
A) A valid argument
B) An error in reasoning
C) A true statement
D) A mathematical proof
Answer: B

22. Ad hominem fallacy attacks:
A) The argument
B) The evidence
C) The person
D) The conclusion
Answer: C

23. Which fallacy appeals to popular opinion?
A) Ad hominem
B) Ad populum
C) Straw man
D) False cause
Answer: B

24. A hasty generalization is based on:
A) Too much evidence
B) Insufficient evidence
C) Perfect logic
D) Mathematical proof
Answer: B

25. The straw man fallacy involves:
A) Strengthening an opponent's argument
B) Misrepresenting an opponent's argument
C) Agreeing with everyone
D) Using valid logic
Answer: B

26. Circular reasoning is when:
A) The conclusion proves the premise
B) The premise assumes the conclusion
C) Arguments are valid
D) Logic is sound
Answer: B

27. False cause fallacy assumes:
A) No relationship exists
B) Correlation equals causation
C) All causes are true
D) Effects precede causes
Answer: B

28. Appeal to authority becomes a fallacy when:
A) The authority is an expert
B) The authority is not an expert in the relevant field
C) Evidence is provided
D) Logic is used
Answer: B

29. Which of the following is an informal fallacy?
A) Valid syllogism
B) Mathematical proof
C) Ad hominem attack
D) Logical deduction
Answer: C

30. Fallacies of ambiguity arise from:
A) Clear language
B) Unclear language
C) Valid reasoning
D) Sound arguments
Answer: B
