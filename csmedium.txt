## Section 1: Operating Systems

1. Operating Systems (Medium)
1. In virtual memory, when a page fault occurs and all frames are occupied, which page replacement algorithm selects the page that has not been used for the longest time?
A) FIFO
B) LRU
C) Optimal

A) First Come First Served
B) Round Robin
C) Priority Scheduling
D) Shortest Job First
Answer: C

3. In the OSI model, which layer is responsible for end-to-end error recovery and flow control?
A) Network Layer
B) Transport Layer
C) Data Link Layer
D) Session Layer
Answer: B

4. For deadlock to occur, which of the following conditions must ALL be present?

B) Paging
C) Swapping
D) Caching
Answer: B

7. In process scheduling, what is the main advantage of Shortest Job First (SJF) algorithm?
A) Easy to implement

D) No starvation

8. Which OSI layer handles routing and logical addressing?
A) Physical Layer
B) Data Link Layer
C) Network Layer
D) Transport Layer
Answer: C

9. What type of interrupt has the highest priority?
A) Software interrupt
B) Hardware interrupt
