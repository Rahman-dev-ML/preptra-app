## Section 1: Group Theory Basics

1. Group Theory Fundamentals (Easy)
1. A group (G, *) must satisfy all properties EXCEPT:
A) Closure
B) Associativity
C) Commutativity
D) Identity element
Answer: C

2. The order of a group G is:
A) The smallest element
B) The number of elements in G
C) The identity element
D) The inverse operation
Answer: B

3. In the symmetric group S₃, the total number of elements is:
A) 3
B) 6
C) 9
D) 12
Answer: B

4. A subgroup H of G must contain:
A) All elements of G
B) The identity element of G
C) Only prime order elements
D) Exactly half the elements
Answer: B

5. The kernel of a homomorphism is always:
A) Empty
B) The whole group
C) A normal subgroup
D) A cyclic group
Answer: C

6. A group of prime order p is always:
A) Abelian
B) Non-abelian
C) Infinite
D) Simple and cyclic
Answer: D

7. The center Z(G) of a group G contains:
A) Elements that commute with all elements
B) Only the identity
C) Half the elements
D) No elements
Answer: A

8. In a cyclic group of order n:
A) All elements have order n
B) There exists an element of order n
C) No element has order n
D) All elements are identity
Answer: B

9. The identity element in a group is:
A) Not unique
B) Its own inverse
C) Always 0
D) Always 1
Answer: B

10. Two groups with the same order are:
A) Always isomorphic
B) Never isomorphic
C) Not necessarily isomorphic
D) Always equal
Answer: C
